===========================================
TROUBLESHOOTING
===========================================

Problem: Still not receiving SMS after setup
Solution:
1. Check sms_log.txt for error messages
2. Verify Auth Key is correct (no extra spaces)
3. Check MSG91 dashboard for balance
4. Try using 'MSGIND' as sender ID

Problem: "Invalid Auth Key" error
Solution:
- Double-check the key in PHP file
- Make sure no quotes are missing
- Regenerate key from MSG91 dashboard

Problem: "Insufficient balance"
Solution:
- Add credits to MSG91 account
- Or use free test credits first
